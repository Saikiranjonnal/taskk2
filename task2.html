IMAGE TAG

  • img tag in html is used to embed the image into web page
  • src (source) specifies the url of the image fil
  • alt provides a text description of the image incase the image is not displayed
  • we can also add width height title border attribiutes to the tag

ABSOLUTE PATH

  • an absolute path is a complete url that includes the protocol(https/http),domain name,file path
  • it points to a specific location
  • used for linking external resources
  • not dependent on the current file location

RELATIVE PATH

  • A relative path is a partial URL that starts from the current file's location. It's relative to the current directory.
  • Used for linking internal resources
  • Dependent on the current file location
  • Faster and more efficient than absolute paths
